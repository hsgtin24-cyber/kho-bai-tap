Phân tích bài toán

Để trả lời các truy vấn tổng XOR trên đoạn [L, R] trong $O(1)$, ta sử dụng mảng cộng dồn (Prefix XOR). Gọi pref[i] là XOR từ phần tử $1$ đến $i$. Khi đó, XOR trên đoạn [L, R] được tính bằng pref[R] ^ pref[L-1].

Code mẫu (Python)

import sys

def solve():
    input = sys.stdin.read
    data = input().split()
    if not data: return
    
    n, q = int(data[0]), int(data[1])
    arr = [int(x) for x in data[2:n+2]]
    
    # Xây dựng mảng Prefix XOR
    pref = [0] * (n + 1)
    for i in range(n):
        pref[i+1] = pref[i] ^ arr[i]
        
    # Trả lời truy vấn
    idx = n + 2
    out = []
    for _ in range(q):
        l, r = int(data[idx]), int(data[idx+1])
        # Chuyển đổi về 0-based index nếu đề bài dùng 1-based
        ans = pref[r] ^ pref[l-1]
        out.append(str(ans))
        idx += 2
        
    print('\n'.join(out))

if __name__ == '__main__':
    solve()
